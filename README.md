# Yankai Huang

Source for [hyk813.github.io](https://hyk813.github.io/), a personal homepage and technical blog built with the [Chirpy][chirpy] Jekyll theme.

## Writing

Posts live in `_posts` and use standard Markdown with Jekyll front matter.

```shell
bundle exec jekyll serve
```

## Deployment

The site deploys to GitHub Pages from the workflow in `.github/workflows/pages-deploy.yml`.

[chirpy]: https://github.com/cotes2020/jekyll-theme-chirpy/
